# Office Meal App — quick start

## Run locally

```powershell
cd office-meal-app
npm install
npm run dev
```

- **Screens:** http://localhost:5173  
- **API:** http://localhost:3001/health  

## Deploy online (easiest — Render, free)

**Full guide:** [docs/DEPLOY-RENDER.md](./docs/DEPLOY-RENDER.md)

1. Push this folder to **GitHub**
2. [render.com](https://render.com) → **Blueprint** → connect repo
3. Get URL like `https://office-meal-app.onrender.com`
4. Share with manager / employees

No AWS setup required.
